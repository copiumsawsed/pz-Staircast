description=[h2]Q/A[/h2]
description=[h3]Why ZombieBuddy is required?[/h3]
description=This is a Java mod and ZombieBuddy is a Java mod loader. What this mod does is not achievable in a standard mod, and while it is possible to make a Java mod by replacing the game's class files, such mods tend to break even on minor game updates. The mod loader solves this problem by injecting code at runtime.
description=[h3]Can I verify what the mod does and build it myself?[/h3]
description=Yes, the source code and build instructions are on GitHub. See below.
description=[h3]Multiplayer support?[/h3]
description=Yes. [url=https://steamcommunity.com/workshop/filedetails/?id=3619862853]ZombieBuddy[/url] must be installed manually before you join a server.
description=[h3]Is it safe to add to/remove from an existing save?[/h3]
description=Yes, it's totally safe.

Installation requires manual steps, please follow its instructions carefully. Another reason might be that you didn't enable the mod for your save. If that's not the case for you, please report it.
